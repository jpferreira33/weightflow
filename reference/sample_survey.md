# Example survey sample for weightflow

A stratified household sample drawn from `population`, with unequal
design base weights, an unknown-eligibility flag and a response
indicator. Frame auxiliaries (region, sex, age) are known for all units;
survey outcomes (income, employed) are observed only for respondents.
Generated by `data-raw/weightflow_data.R`.

## Usage

``` r
sample_survey
```

## Format

A data frame with one row per sampled person and the columns:

- person_id, household_id:

  identifiers

- region, sex, age:

  frame auxiliaries, known for all units

- pw:

  design base weight (inverse sampling fraction)

- unknown_elig:

  1 if eligibility is unknown

- responded:

  1 if the unit responded

- income, employed:

  survey outcomes; NA for nonrespondents
